# 🚀 DigitalProfitX — High-Performance Agency Landing Page

**DigitalProfitX** is a premium, ultra-luxury, and cinematic digital marketing agency landing page. Built with **React** and powered by **GSAP (GreenSock Animation Platform)**, this project features high-end architectural animations, fluid micro-interactions, and a modern glassmorphism design tailored for performance-driven brands.

---

# 💎 Elite Features & Micro-Interactions

* **Dynamic 3D Perspective Logo:** Features a 360° vertical axis flip (`rotateY`) on hover with hardware-accelerated depth tracking.
* **Luxury Text-Rolling Navbar:** Utilizes dual-layered masking and a custom Cubic-Bezier curve (`ease-[cubic-bezier(0.76,0,0.24,1)]`) for a smooth, vertical rolling hover effect.
* **Cinematic Hero Entrance:** Line-by-line text splitting accompanied by smart blur-ins, smooth scaling call-to-actions, and precise overlapping timelines.
* **Ambient Floating Cards:** Interactive high-fidelity metric widgets (+280% Revenue, 200+ Clients, 4.8x ROAS) running on an infinite, independent physics-less loop.
* **Dynamic Services Ecosystem:** Clean grid showcasing tailored performance solutions from SEO to multi-channel e-commerce management.
* **Social Proof Showcase:** High-converting feedback masonry layout emphasizing high performance, conversion rate optimization, and measurable brand scaling.
